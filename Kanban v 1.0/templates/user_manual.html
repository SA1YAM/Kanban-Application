Welcome to Simple Kanban Board



Manage your tasks by using virtual Kanban App which is very easy to use.




Have a quick grasp of the functionality given below for better understanding.

    • Used for tracking tasks.
    • User can have multiple lists.
    • User can add one or more cards to a list.
    • Each card will display Title,Content,Started date,Deadline Date and it's status.
    • At the bottom of the card the status of the card will be shown i.e completed or Incomplete.
    • Each Card can be moved from one list to another list.
    • Summary - Shows how the user is performing across lists based on the completed flag, time when it completed, it also shows graphs.
    • Summary - Date of completion for completed tasks will be shown as label value for that card in the bar graph.
    • Summary page will also show the number of completed, Inprogress and Deadline Crossed cards/tasks.
